# InvestMZ Wallet

Carteira digital para Moçambique com M-Pesa e e-Mola.

## Deploy 1-Click no Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/seu-usuario/investmz-wallet)

Ou manual:
1. Fork este repo
2. Render.com > New > Blueprint > conecta repo
3. Adiciona variáveis: E2PAYMENTS_KEY, DATABASE_URL
4. Deploy

## Configurar e2Payments
Callback URL: https://seu-app.onrender.com/webhook/e2payments
